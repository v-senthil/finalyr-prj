About Project

Crowd management is the organised and substantiated planning and the direction given to the orderly progress of events where large groups of people gather together. As a part of crowd management, measures can be taken to direct or limit the behaviour of groups of people.


A fully automated System built over the existing camera system in auditoriums, conference halls, large meeting place or any place with multiple exits and number of people attending the events. This new system play a vital role in Crowd Management in the exits, Mask Detection, Number of people going in and out the place, Emotion Analysis, and Social Distancing System

Dashboard

Responsive Dashboard which can not be obtained in any online Dashboard Systems. Simple, Elegant, and Precise.

Crowd Management System

Over 1000 people attend a normal event but Managing the crowd is a big task. Our system makes it easy and simple.

Mask Detection System

As Covid has taken a lot of people lifes, and is spreading in a rapid speed, masks have become a part of our lives now.

People In and Out

Number of people attending the event could vary every time. Even if people buy tickets, some would fail to attend the events. Count people in and out the place would give the exact count.

Emotion Analysis

People can give feedbacks in a form which who knows is correct. Live Emotion Analysis during the event will give more precise feedback of that specific event.

Social Distancing

COVID 19, has affected a lot of lifes. Social Distancing has become a regular practice in important places in our world right now. Even though it's new normal!

Live-Stream

This whole System is built over the existing camera system in that specific event place. Live Streaming of all cameras which can be accessed all over the world is possible with this system.

Reports

All the data is stored in Database, which then can be downloaded as Excel and CSV format for further analysis.

Notification

If there is any danger like no masks or people are overcrowded or no social distance is maintained, the user will receive a message and an Email automatically (or) manually.

File Storage

If there is any data or files to be uploaded to the cloud or the server side, our system allows that too.

Documentation

A full fledged Documentation along with API docs is documented.

API Calls

As this system is Open-Source, people can use our API's to obtain only a certain amount of data using our algorithms.
